# Warrior RPG System 🛡️

Це консольний застосунок на мові **C#**, створений для демонстрації принципів ООП (наслідування та інкапсуляція).

## Про проєкт
Програма імітує систему персонажів у грі. Вона дозволяє створювати різних воїнів, кожен з яких має свої особливості розрахунку характеристик та унікальні вміння.

## Ієрархія класів
1. **Warrior** — базовий клас.
   - Поля: Здоров'я, Броня, Урон.
   - Метод `TakeDamage`: зменшує здоров'я з урахуванням броні.
2. **Knight** — лицар, що може молитися (`Pray`) для збільшення броні.
3. **Barbarian** — варвар, чия атака залежить від швидкості. Вміє використовувати бойовий крик (`Shout`).

## Як запустити
1. Встановіть [.NET SDK](https://dotnet.microsoft.com/download).
2. Склонуйте репозиторій.
```
git clone https://github.com/Kernel-vlad/CSharp-BaseProject.git
```
4. Запустіть команду `dotnet run` у папці проєкту.

## Приклад роботи
При отриманні 150 одиниць шкоди лицарем із бронею 20, його здоров'я розраховується як:
$100 - (150 - 20) = -30$
